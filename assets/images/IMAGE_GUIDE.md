# Image Guide for Trekkers & Tourers Website

## Required Images

This folder contains all images used on the Trekkers & Tourers travel website. Below is the mapping of required images and their usage:

### Hero Section Image
- **Filename**: `hero-trek.jpg`
- **Usage**: Homepage hero section (mountain/trek with light rays)
- **Dimensions**: 800x600px (recommended)
- **Location in HTML**: index.html, line ~37

### Trip Card Images (Signature Experiences)
- **Image 1 - Mountains Trek**
  - **Filename**: `trip-mountains.jpg`

  - **Dimensions**: 800x600px
  - **Location in HTML**: index.html, line ~48

- **Image 2 - Beach Getaway**
  - **Filename**: `trip-beach.jpg`
  - **Usage**: Coastal Beach Escape card
  - **Dimensions**: 800x600px
  - **Location in HTML**: index.html, line ~55

- **Image 3 - Waterfall Trail**
  - **Filename**: `trip-waterfall.jpg`
  - **Usage**: Waterfall & Forest Trail card
  - **Dimensions**: 800x600px
  - **Location in HTML**: index.html, line ~62

### Why Us Section Image
- **Filename**: `why-us.jpg`
- **Usage**: "Why Travel With Us" section (happy group of travelers)
- **Dimensions**: 800x600px
- **Location in HTML**: index.html, line ~82

## How to Upload Images

1. Go to the GitHub web interface
2. Navigate to `/assets/images/` folder
3. Click "Add file" > "Upload files"
4. Drag and drop or select your image files
5. Ensure filenames match exactly as listed above
6. Commit the changes

## Image Optimization Tips

- Compress images before uploading (recommended size: < 500KB per image)
- Use JPG format for photographs
- Use PNG format for graphics with transparency
- Maintain 4:3 aspect ratio for consistent card layouts

## Additional Images

You can add more images as needed (up to 13 total as mentioned). Examples:
- `hero-secondary-1.jpg`
- `testimonial-image-1.jpg`
- `gallery-1.jpg`
- etc.

After adding new images, update `index.html` to reference them.
